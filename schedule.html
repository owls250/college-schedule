Schedule

How to use

  1. Copy your schedule from your syllabus/schedule pdf to Excel/Google Sheets
  2. Save the file/ download it as a csv with the class name as the file name
  3. Repeat for each class, one file per class
  4. Use the browse button below to select each csv file
  5. Click Submit to submit all schedules
